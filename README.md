# LLMs-Hallucination-Detection-and-Correction-in-LLMs
本项目通过三个独立实验，完成了针对LLMs幻觉率问题的分析与矫正方法研究。3个实验分别为不同超参数数量的LLMs幻觉率比较实验、相同超参数约束下的开源闭源LLMs幻觉率对比实验、以及降低幻觉率的提示策略以及RAG消融实验。项目采取的数据集为HaluEval。项目实现了小规模模型的本地化部署以及优化任务，同时完成了商业API端口的接入实验，为大学新生以及LLM初学者提供项目参考。
项目实验文件分为data，code以及result三个文件夹，其中result为本实验的参考结果。data为本地数据集，code为代码实现。report为全英文报告。
